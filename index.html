Smart Farmer

In this project we have developed a mobile application using which a farmer can monitor the temperature, humidity, pressure and soil moisture parameters along with weather forecasting details. Based on these details he can water the crops by controlling the motors through the app and the app gives an alert message if temperature or humidity goes beyond a threshold value.

DRONEO

The system is an AI-based solution for automatic crop weeding recognition and selective spraying. It uses computer vision and machine learning to detect weeds and applies herbicides only where needed, reducing chemical usage and environmental impact. By integrating with agricultural machinery, such as tractors or drones, the system operates efficiently and autonomously, improving the overall productivity of crop management while minimizing the labor required for manual weeding.
